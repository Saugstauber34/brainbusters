# BrainBusters 🎮

BrainBusters ist ein Quizspiel, das in Python entwickelt wurde.  
Spieler können ihr Wissen testen, Punkte sammeln und sich mit anderen vergleichen.

---

## 🚀 Funktionen

- 👤 Benutzerregistrierung & Login
- 🧠 Quiz mit verschiedenen Kategorien
- 🌐 Online-Quiz (API)
- 🏆 Rangliste
- 👥 Mehrspielermodus (2 Spieler)
- 🏅 Achievement-System
- 🛠 Admin-Backend zur Verwaltung von Fragen
- 🧪 Automatisierte Tests mit pytest

---

## 🛠 Technologien

- Python
- SQLite (Datenbank)
- Open Trivia API
- pytest (Tests)
